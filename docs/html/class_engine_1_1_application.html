Detailed Description

Central engine object that owns and orchestrates every subsystem.

Application is the entry point for both the editor and the runtime. It creates the window, initialises the renderer, audio, physics and scripting systems, then runs the main loop.

Typical usage (minimal runtime):

Engine::Application app;
app.Init();
app.Running(); // blocks until the window is closed
app.Shutdown();
