Detailed Description

Abstract base class for every game system (MovementSystem, RenderSystem, …).

A System contains logic only – it never owns component data. The SystemManager automatically populates mEntities with every entity whose component signature matches the system's registered signature.

Derived systems typically override Update(), Render(), or a custom tick method and iterate over mEntities to process matching components.
