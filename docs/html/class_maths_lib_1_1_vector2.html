Detailed Description

template<typename T>
class MathsLib::Vector2< T >

Generic 2D vector with common math operations.

Template Parameters
TScalar type (e.g. float, double, int).

Provides arithmetic operators, static utility methods (Dot, Lerp, Distance…) and direction constants (up, down, left, right, zero, one).
