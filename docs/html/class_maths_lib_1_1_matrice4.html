Detailed Description

template<typename T>
class MathsLib::Matrice4< T >

A 4×4 homogeneous transformation matrix.

Template Parameters
TScalar type (e.g. float, double).

Supports projection matrices (perspective, orthographic), TRS composition, inversion, transposition and quaternion extraction.
