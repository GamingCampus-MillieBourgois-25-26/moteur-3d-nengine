Detailed Description

template<typename T>
class MathsLib::Quaternion< T >

Represents a rotation in 3D space as a unit quaternion.

Template Parameters
TScalar type (typically float or double).

Provides construction, normalisation, SLERP/LERP interpolation, Euler-angle conversion, and static factory methods identical in spirit to Unity's Quaternion API.
