Detailed Description

Forwards linear forces and torques from Force components to btRigidBody.

Required components: Force (Force::active == true)

Each physics tick:

  • ForceMode::Force → calls btRigidBody::applyCentralForce() / applyTorque() every frame (force remains active).
  • ForceMode::Impulse → calls btRigidBody::applyCentralImpulse() / applyTorqueImpulse() once, then resets the Force component and clears the active flag.

PhysicsBodySystem owns the btRigidBody instances; this system only reads them.
