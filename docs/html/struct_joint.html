Detailed Description

Constrains the motion between two rigid-body entities.

The JointSystem reads this component, retrieves the two btRigidBody pointers from the PhysicsBodySystem, creates the appropriate btTypedConstraint, and registers it with the Bullet dynamics world.

Hinge limits (lowerLimit / upperLimit) are in radians. Setting breakingForce = 0 makes the joint unbreakable.

Note
Both entities must own a Collider so their rigid bodies exist before the joint is initialised.
