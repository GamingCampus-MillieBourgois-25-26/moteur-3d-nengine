Detailed Description

template<typename T>
class MathsLib::Vector3< T >

Generic 3D vector with common math operations.

Template Parameters
TScalar type (e.g. float, double).

Provides arithmetic operators, dot/cross products, interpolation, and direction constants (up, down, forward, back, left, right, zero, one).
