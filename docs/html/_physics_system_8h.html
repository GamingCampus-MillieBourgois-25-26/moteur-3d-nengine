Detailed Description

Forward-includes for the Bullet Physics library.

This header is a minimal convenience include that pulls in the Bullet main header. Actual physics functionality is implemented in PhysicsBodySystem, ColliderSystem, ForceSystem, JointSystem, TriggerSystem and CharacterControllerSystem.
