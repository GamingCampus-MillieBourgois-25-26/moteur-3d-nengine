Detailed Description

Manages the collection of Scenes and tracks the active one.

Application owns one SceneManager. Scenes are created via CreateScene() and identified by name. Only one scene is "active" at a time; Application::Running() drives the active scene's Update() and Render().

Usage:

Scene* main = sceneManager.CreateScene("Main", &renderer, &scripts);
sceneManager.SetActiveScene("Main");
Scene* active = sceneManager.GetActiveScene();
