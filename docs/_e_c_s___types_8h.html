Unique index assigned to a component type when it is registered.

◆ Entity

using Entity = std::uint32_t

Unique identifier for a living entity.

An entity is simply an integer ID. Example: Entity player = 1; Entity enemy = 2;

◆ Signature

using Signature = std::bitset<MAX_COMPONENTS>

Bitmask describing which components an entity owns.

Each bit corresponds to a ComponentType index. Example: 0b101 → owns component 0 (Transform) and component 2 (MeshRenderer).
