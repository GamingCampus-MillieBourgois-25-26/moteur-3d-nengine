Detailed Description

Defines the physical properties of a playable kinematic character.

The CharacterControllerSystem reads this component to create a btKinematicCharacterController backed by a btPairCachingGhostObject and a btCapsuleShape inside the Bullet dynamics world.

Each frame the system:

  1. Applies walkDirection and jump to the controller.
  2. Synchronises the resulting position back into the entity's Transform.

Workflow:

  • Set walkDirection every frame from gameplay code (input system).
  • Set jump = true to trigger a jump; the system consumes it.
  • Read onGround to determine whether the character can jump again.
