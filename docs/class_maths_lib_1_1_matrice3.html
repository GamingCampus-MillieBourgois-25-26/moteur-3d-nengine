Detailed Description

template<typename T>
class MathsLib::Matrice3< T >

A column-major 3×3 matrix.

Template Parameters
TScalar type (e.g. float, double).

Provides basic arithmetic operators and an equality test. Used for 2D linear transformations and 3D rotation matrices.
