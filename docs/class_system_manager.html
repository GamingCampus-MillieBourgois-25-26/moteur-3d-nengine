Detailed Description

Registers systems, assigns component signatures, and keeps each system's entity set up-to-date.

Role:

  • Register systems (MovementSystem, RenderSystem, …).
  • Associate a required component Signature with each system.
  • When an entity's signature changes, add or remove it from each system whose required signature is a subset of the entity's signature.

This is the "wiring" layer that connects entities to the systems that should process them.
